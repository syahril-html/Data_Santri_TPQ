Sistem Manajemen Data Santri TPQ

Aplikasi dashboard web full-stack sederhana untuk mengelola data santri.

Fitur Utama
* **Dashboard CRUD Lengkap**: Tambah, Baca, Edit, dan Hapus data santri secara real-time.
* **Struktur Data 12 Kolom**: Mengelola informasi detail mulai dari Nama, NIK, TTL, Kelas, hingga data orang tua (Ayah/Ibu).
* **Antarmuka Responsif**: Tabel data yang mendukung scroll horizontal dan modal form yang melayang (overlay) di tengah layar.
* **Styling Modern**: Header tabel dengan tombol "+ Tambah Santri" di pojok kanan yang rapi dan fungsional.

Teknologi yang Digunakan
* **Backend**: Cloudflare Workers (JavaScript).
* **Database**: Cloudflare D1 (SQL-based).
* **Frontend**: HTML5, CSS3 (Flexbox & Grid), dan Vanilla JavaScript.
* **Tooling**: Wrangler CLI untuk deployment dan local development.
